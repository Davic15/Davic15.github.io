AppShareBooks
(Group project at university). Project developed for Mobile Application Development at University. The focus of the project was to borrow and/or lend books.
Android Project - AppShareBooks

An App that helps people to share books. The app allows users to sign up and set up a basic profile. Users can make books available for sharing, providing all the relevant information (stored in a database). Users can search for shared books and get in contact, via the app, with the book owner in order to arrange the withdrawal and successive return. As a consequence of each sharing, users' reputation must be updated.
